# Wolf Conservation Project

Bioinformatics pipeline to analyse modern wolf genomes  

*Updated by Evelyn Todd, February 2026*

**General Information**

This repository contains the code for population genetic analysis of European wolves, including:

- Filtering
- PCA
- Admixture
- Dstats
- F4-ratio tests
- Orientagraph
- Genetic Diversity

*This project is ongoing and code will be updated*
